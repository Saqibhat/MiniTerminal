# Terminal Shell i Go

En kommandolinje-shell implementert i Go som støtter grunnleggende filsystem-operasjoner.

## Funksjoner
- Mappenavigasjon (cd)
- Filhåndtering (ls, cat, head, tail)
- Opprett/slett filer og mapper (mkdir, rm, opprett)
- Kommandohistorikk
- Feilhåndtering

## Teknologier
- Go
- OS pakke for filsystem-interaksjon
- Bufio for brukerinndata

## Hvordan kjøre
```bash
go run main.go
```

## Kommandoer
- `avslutt` - Avslutt programmet
- `cd [sti]` - Bytt mappe
- `ls [sti]` - List filer
- `hjelp` - Vis alle kommandoer
